# Knowledge-Vault

## What This Is

Central repository for organizational knowledge, documentation, and reference materials. Stores long-form documentation, guides, and resources that don't belong in individual project repos.

## Core Value

Persistent organizational memory that survives across projects and sessions.

## Requirements

### Validated

- Centralized documentation storage
- Cross-project reference materials
- Knowledge organization structure

### Active

- Continuous knowledge accumulation

### Out of Scope

- Project-specific documentation (stays in project repos)
- Automated aggregation (handled by ops-core)

## Context

**Current State:** Active knowledge repository.

**Structure:** Organized by topic and project references.
